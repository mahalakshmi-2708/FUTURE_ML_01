## 🎯 Objective
To build a machine learning model that forecasts future sales using historical business data.

---

## 📂 Dataset
The dataset contains historical sales records including order date, sales amount, and other business-related attributes.

---

## ⚙️ Tools & Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  

---

## 🔍 Approach
1. Loaded and explored the dataset  
2. Selected relevant columns (YEAR_ID, MONTH_ID, SALES)  
3. Aggregated monthly sales using groupby  
4. Created a time-based feature for forecasting  
5. Built a Linear Regression model  
6. Predicted future sales values  
7. Visualized actual vs predicted sales  

---
